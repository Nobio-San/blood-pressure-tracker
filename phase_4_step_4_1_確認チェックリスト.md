# Step4-1 デザインの洗練 - 確認チェックリスト

## 画面表示

- [ ] 入力フォーム：カードで区切られ、見出し・本文の階層が分かる
- [ ] 一覧セクション：カードで区切られ、記録が読みやすい
- [ ] グラフセクション：カードで区切られ、フィルタ・グラフが配置されている
- [ ] モーダル（画像拡大等）：オーバーレイ・閉じるボタンが視認できる

## レスポンシブ

- [ ] スマホ幅（360〜430px）：横スクロールが出ない
- [ ] タブレット幅（768px前後）：余白が適切、フィルタ等が横並び
- [ ] PC幅（1024px+）：最大幅960pxで中央配置、情報が散らばらない

## アクセシビリティ

- [ ] キーボードTab移動でフォーカスが視認できる（フォーカスリング表示）
- [ ] ボタン・入力のタップ領域が48px以上
- [ ] 最小フォント16px以上（補助テキストは14px）
- [ ] アイコンのみボタンに `aria-label` がある

## 機能確認

- [ ] 入力 → 保存 → 一覧更新 → グラフ更新 が動作する
- [ ] フォームのバリデーションエラー表示が分かりやすい
- [ ] disabled状態（送信中など）の見た目が分かる
